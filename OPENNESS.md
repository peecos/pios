# Open by Design

PIOS is intended to be used, shared, improved, implemented, and forked openly.

## Documentation and Media

The PIOS 2.0 framework documentation in this repository is available under
Creative Commons Attribution 4.0 International (CC BY 4.0). You may read,
share, copy, translate, adapt, and build on it, including for commercial,
educational, research, community, and independent implementation work.

Please give attribution. A simple form is:

```text
Based on PIOS 2.0 by Valto Loikkanen / peecos.
Used under CC BY 4.0.
```

See [LICENSE.md](LICENSE.md) for the complete licence text.

## Independent Implementations and Forks

Independent implementations, applications, agents, services, compatible
ecosystems, translations, and forks are welcome. No hosted provider has an
exclusive right to implement PIOS.

When interoperating with PIOS material, resolve and describe the actual
compatibility level honestly. Do not imply broader support than an
implementation provides.

## Names, Logos, and Endorsement

Open documentation does not make an independent project official. You may say
that work is based on PIOS, implements PIOS concepts, or is PIOS-compatible
when that is accurate. Do not claim to be the official PIOS initiative, to be
officially endorsed without permission, or to own the project identity.

This limited boundary protects clarity and trust. It does not prevent
independent development or forks under another name.

See [TRADEMARKS.md](TRADEMARKS.md) for the full guidance.
